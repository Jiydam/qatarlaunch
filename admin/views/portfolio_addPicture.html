Notes on uploading images

You can upload any size of image so long as each side is greater than 700 pixels.

If you upload an image where has any side is greater than 1200 pixels, it will be scaled back down to 1200 pixels with the aspect ratio maintained.

You can upload more than one image at a time.

You can only upload .jpg files.
